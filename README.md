# 💰 Future Finance AI

Future Finance AI is an AI-powered web app that helps users make smarter financial decisions. It analyzes income, expenses, and savings in real-time to generate a Financial Health Score and provide personalized advice.

Using structured financial logic combined with Grok AI, the system delivers purchase guidance, risk warnings, and improvement tips—helping users understand the long-term impact of their financial choices.

Built with Next.js and deployed on Vercel, the platform is designed to be scalable, intelligent, and accessible for users at any income level.
